# Data Artifacts

This directory is for generated local/operator data artifacts that should not be
tracked in git by default.

Current intended use:

- `vps_daily_reports.db`
  Normalized SQLite history generated by
  `scripts/vps_daily_report.py`

The canonical tracked artifacts remain:

- generated Markdown reports under `docs/`
- archive metadata and zip bundles under `docs/archive/`
